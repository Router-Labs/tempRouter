Confidential-inference rail · Tempo · MPP · Phala Intel TDX

Pay an LLM only after you can
prove it never saw your prompt.

tempRouter is infrastructure for confidential agent inference on Tempo. Your agent verifies, with Intel DCAP, that the prompt runs inside a real Phala Intel TDX enclave that can't read it — then pays per response-chunk in pathUSD. The relay is blind: ciphertext only, no key. Ships as an SDK, CLI, and MCP server, so you drop it into any agent.

● SDK · CLI · MCP ● verify-before-pay ● per-chunk MPP metering ● no API key · no signup ● blind relay
Why

Payment was the easy half.

MPP already does per-chunk stablecoin billing. tempRouter ships the half MPP left open — provable confidentiality — a payable inference endpoint whose prompt is E2E-encrypted to a real Intel TDX enclave you verify before you pay.

Verify before you pay

The agent runs Intel DCAP on the live enclave quote before signing the first voucher — fail the check and it signs zero vouchers. A bad enclave can't decrypt the prompt anyway, so money only moves for inference you proved is confidential.

Two-phase Intel DCAP verify

Pre-pay: genuine TDX hardware + UpToDate TCB + code measurement. Post-pay receipt: ed25519 over the exact ciphertext + key binding. Auditable on the wire.

Blind relay

Prompts are E2E-encrypted (Arcium RescueCipher + X25519) to a key that exists only inside the enclave. tempRouter never holds a key and never sees plaintext.

Client-side secret/PII policy

The agent detects API keys, private keys, JWTs and PII locally and forces the private lane — the right call because a blind relay can't classify what it can't read.

MPP session metering

MPP session vouchers over SSE: pay per response-chunk in pathUSD, settled in ~2 on-chain txs. Bills one charge per inference; multi-chunk metering is supported.

Discoverable + payable

OpenAI-shaped request, MPP 402 flow, and an OpenAPI discovery doc so any agent or registry (mpp.dev/services, MPPScan) can find and pay the endpoint.

Flow

How an agent uses it.

  1. Detect — the agent's policy flags a secret/PII payload and forces the private lane.
  2. Verify — GET /tee/attestation, run Intel DCAP on the quote. Fail → sign zero vouchers, refuse to pay.
  3. Encrypt — encrypt the prompt to the enclave's X25519 key. Plaintext never leaves the agent.
  4. Pay — POST /v1/chat/completions/stream → 402 → open an MPP session on Tempo, pay per response-chunk.
  5. Decrypt — reassemble the ciphertext chunks and decrypt locally. Only the agent and the attested enclave ever see plaintext.
Integrate

Drop it into your agent.

tempRouter is infrastructure, not a destination. The same verify-before-pay lane ships at four altitudes — an SDK, a CLI, an MCP server, and an agent skill. Each does the whole dance: verify the enclave, encrypt, pay per chunk in pathUSD, decrypt. Pick the one that fits.

// @temprouter/sdk — the whole lane in one call
import { TempRouter, detectSensitive } from '@temprouter/sdk'

const client = new TempRouter({ serverUrl, account }) // account = funded Tempo wallet

if (detectSensitive(prompt).sensitive) {
  const { answer, paid } = await client.infer(prompt) // verify → encrypt → pay → decrypt
}

SDK

@temprouter/sdk

One client.infer(prompt) call. Bundles @solrouter/sdk encryption + mppx Tempo payment + Intel DCAP verify. A failed gate throws AttestationError and pays zero.

CLI

temprouter

temprouter infer "…" — verify, pay per chunk, decrypt at the terminal. Plus verify (gate only, free) and detect (policy check).

MCP server

stdio · local

Tools private_inference, detect_sensitive, verify_enclave. Runs in your agent's process, so encryption + wallet never leave your trust domain.

Agent skill

entrypoint

Drop the temprouter skill and your agent auto-routes any secret/PII prompt to the private lane instead of a public model host.

# or straight from a shell / agent terminal

Under the hood

The payable endpoint.

All four surfaces speak the same wire protocol. Any MPP-speaking agent can hit it directly — the 402 handshake handles pay → retry.

POST/v1/chat/completions/stream0.0002 pathUSD / response-chunk
Attested private inference, metered per response-chunk over an MPP session (SSE). MPP tempo intent session, currency pathUSD, network Tempo Moderato (chain 42431).
GET/tee/attestationfree · verify before you pay
The live enclave attestation (teeType: INTEL-TDX-PHALA + TDX quote). Verify with Intel DCAP before signing a voucher.
