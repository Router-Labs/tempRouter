Payment was the easy half.

🔍

Verify before you pay

Your agent runs Intel DCAP on the live enclave quote before signing any voucher. Fail → sign zero vouchers. The enclave can't decrypt the prompt anyway.

🔐

Blind relay

The prompt is E2E-encrypted to a key that only exists inside the enclave. tempRouter holds no key — it forwards ciphertext and meters the reply.

💸

Per-chunk MPP metering

Native session over SSE: pay per response-chunk in stablecoin, settled in ~2 on-chain txs via Tempo. No API keys, no subscriptions.

🧠

Client-side secret policy

Your agent detects API keys, private keys, JWTs and PII locally — and forces the private lane. The relay can't classify what it can't read.

📜

Full transparency

Every check is auditable on the wire: DCAP cert chain, key binding, enclave ed25519 signature, quote digest. Nothing taken on trust.

🔌

Discoverable + payable

OpenAPI + llms.txt + agent skill. Any MPP-speaking agent can discover and pay the endpoint — no human in the loop.
